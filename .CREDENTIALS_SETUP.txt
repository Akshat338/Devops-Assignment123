# GitHub Actions Secrets Configuration Guide

## Docker Hub Credentials
- `DOCKERHUB_USERNAME`: akshat919
- `DOCKERHUB_TOKEN`: [YOUR_DOCKER_HUB_TOKEN_HERE]

## EC2 SSH Configuration  
- `VM_HOST`: [YOUR_EC2_PUBLIC_IP]
- `VM_USER`: ubuntu
- `VM_SSH_KEY`: [CONTENTS_OF_AKSHAT_PEM_FILE]
- `VM_PORT`: 22

## How to Add These to GitHub

1. Go to GitHub Repository Settings
2. Navigate to Secrets and variables → Actions
3. Click "New repository secret"
4. Add each secret with the values above

## Important Notes

- **Never commit actual secrets to git**
- **Docker Hub Token** should start with `dckr_pat_`
- **VM_SSH_KEY** should include the full PEM file content (including BEGIN and END lines)
- Keep all secrets secure and private

## For Local Docker Login

```bash
docker login
# Username: akshat919
# Password: [Your Docker Hub token]
```

## For SSH Connections

```bash
# Ensure Akshat.pem has correct permissions
chmod 600 Akshat.pem

# Connect to EC2
ssh -i Akshat.pem ubuntu@YOUR_EC2_IP
```
